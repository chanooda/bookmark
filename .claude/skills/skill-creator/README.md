# Skill Create

스킬은 모듈식 독립 패키지로, 전문 지식, 워크플로우, 도구를 제공하여 클로드의 기능을 확장합니다. 특정 도메인이나 작업을 위한 '온보딩 가이드'라고 생각하면 됩니다. 스킬은 클로드를 범용 에이전트에서 절차적 지식을 갖춘 전문 에이전트로 변모시키며, 이는 어떤 모델도 완전히 보유할 수 없는 능력입니다.

## 스킬이 제공하는 것

전문적인 워크플로 - 특정 도메인을 위한 다단계 절차
도구 통합 - 특정 파일 형식 또는 API 작업 지침
도메인 전문성 - 기업별 지식, 스키마, 비즈니스 로직
묶음 리소스 - 복잡하고 반복적인 작업을 위한 스크립트, 참조 자료 및 자산
